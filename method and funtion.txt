1. Functions (الدوال الجاهزة – Built-in Functions)
دي حاجات بايثون عاملها جاهزة تقدر تستدعيها على طول عشان تعمل شغل معين.
مثال:

python

len("hello")     # ترجع طول النص → 5
type(123)        # ترجع نوع البيانات → <class 'int'>
print("Hi")      # تطبع النص على الشاشة
len(), type(), print() دي Built-in Functions.

2. Methods (الطرق – Methods)
الـ Method هي دالة لكن بتيجي مرتبطة بنوع بيانات معين، وتستدعيها باستخدام النقطة ..
مثال:

python

name = "MOHAMED"
print(name.lower())   # يحول النص لحروف صغيرة → "mohamed"

fruits = ["apple", "banana"]
fruits.append("orange")   # يضيف عنصر للقائمة
print(fruits)             # ["apple", "banana", "orange"]
.lower() دي Method خاصة بالـ strings.

.append() دي Method خاصة بالقوائم (lists).

💡 الخلاصة:

len() و type() → Built-in Functions.

.lower() و .append() → Methods مرتبطة بنوع بيانات محدد.

==================================================================
🟢 أولًا: أشهر الـ Built-in Functions (الدوال الجاهزة)

| الدالة    | الاستخدام                 | مثال                     | النتيجة           |
| --------- | ------------------------- | ------------------------ | ----------------- |
| `print()` | طباعة النصوص أو القيم     | `print("Hello")`         | `Hello`           |
| `len()`   | معرفة طول النص أو القائمة | `len([1, 2, 3])`         | `3`               |
| `type()`  | معرفة نوع البيانات        | `type(3.5)`              | `<class 'float'>` |
| `int()`   | تحويل لقيمة صحيحة         | `int("5")`               | `5`               |
| `float()` | تحويل لقيمة عشرية         | `float("5")`             | `5.0`             |
| `str()`   | تحويل لنص                 | `str(123)`               | `"123"`           |
| `input()` | أخذ إدخال من المستخدم     | `name = input("اسمك؟ ")` | يطبع النص المدخل  |
| `max()`   | أكبر قيمة                 | `max(3, 8, 1)`           | `8`               |
| `min()`   | أصغر قيمة                 | `min([2, 5, 0])`         | `0`               |
| `sum()`   | مجموع القيم               | `sum([1, 2, 3])`         | `6`               |
| `round()` | تقريب رقم                 | `round(3.14159, 2)`      | `3.14`            |

==================================================================
🔵 ثانيًا: أشهر الـ Methods الخاصة بـ Strings (النصوص)

| الـ Method      | الاستخدام                          | مثال                                  | النتيجة           |
| --------------- | ---------------------------------- | ------------------------------------- | ----------------- |
| `.lower()`      | تحويل النص لحروف صغيرة             | `"HELLO".lower()`                     | `"hello"`         |
| `.upper()`      | تحويل النص لحروف كبيرة             | `"hi".upper()`                        | `"HI"`            |
| `.capitalize()` | أول حرف كبير                       | `"hello".capitalize()`                | `"Hello"`         |
| `.title()`      | أول حرف من كل كلمة كبير            | `"hi mohamed".title()`                | `"Hi Mohamed"`    |
| `.strip()`      | إزالة المسافات من البداية والنهاية | `" hi ".strip()`                      | `"hi"`            |
| `.replace()`    | استبدال نص بآخر                    | `"hi mohamed".replace("hi", "hello")` | `"hello mohamed"` |
| `.split()`      | تقسيم النص لقائمة                  | `"a,b,c".split(",")`                  | `["a", "b", "c"]` |
| `.join()`       | دمج قائمة نصوص                     | `",".join(["a","b","c"])`             | `"a,b,c"`         |
| `.find()`       | البحث عن كلمة وإرجاع موقعها        | `"mohamed".find("a")`                 | `4`               |
========================================================
🟠 ثالثًا: أشهر الـ Methods الخاصة بـ Lists (القوائم)

| الـ Method   | الاستخدام                           | مثال                            | النتيجة         |
| ------------ | ----------------------------------- | ------------------------------- | --------------- |
| `.append()`  | إضافة عنصر للقائمة                  | `nums = [1, 2]; nums.append(3)` | `[1, 2, 3]`     |
| `.insert()`  | إدخال عنصر في مكان معين             | `nums.insert(1, 99)`            | `[1, 99, 2, 3]` |
| `.remove()`  | إزالة عنصر بالقيمة                  | `nums.remove(99)`               | `[1, 2, 3]`     |
| `.pop()`     | إزالة عنصر بالمكان (افتراضي الأخير) | `nums.pop()`                    | `[1, 2]`        |
| `.sort()`    | ترتيب القائمة                       | `nums.sort()`                   | `[1, 2, 3]`     |
| `.reverse()` | عكس ترتيب العناصر                   | `nums.reverse()`                | `[3, 2, 1]`     |
| `.count()`   | عدد مرات تكرار قيمة                 | `nums.count(2)`                 | `1`             |
=======================================================
